# Viral-YT-Architect: Your AI YouTube Producer 🎬
**Viral-YT-Architect** is a high-performance Python automation tool designed to bridge the gap between a raw idea and a viral-ready production plan. By leveraging advanced Generative AI, this tool automates the heavy lifting of content strategy, allowing creators to focus on what matters: filming and connecting with their audience.
### 🚀 Developed By: Hammad Virk 🇵🇰
## 💡 Value Proposition
Most creators fail not because of their filming quality, but because of **Title-Thumbnail Mismatch** and **Weak Hooks**.
Viral-YT-Architect eliminates "Writer's Block" by instantly generating:
 1. **Psychology-Backed Titles:** Three variants designed to maximize Click-Through Rate (CTR).
 2. **Engagement-Optimized Script:** A full script structure (Hook, Value-Packed Body, Call-to-Action).
 3. **Visual Strategy:** Five descriptive AI prompts for Midjourney or DALL-E to create world-class thumbnails or B-roll.

## ⚡ How to Run (1-2-3 Steps)
### 1. Clone & Install
```bash
git clone [https://github.com/yourusername/Viral-YT-Architect.git](https://github.com/yourusername/Viral-YT-Architect.git)
cd Viral-YT-Architect
pip install -r requirements.txt

```
### 2. Set Up Your API Key
Get your Gemini API Key from Google AI Studio and set it in your environment:
```bash
# Windows
set GOOGLE_API_KEY=your_actual_key_here
# Mac/Linux
export GOOGLE_API_KEY=your_actual_key_here

```
### 3. Launch the Architect
```bash
python yt_architect.py

```
Follow the interactive terminal prompts to generate your viral content!
